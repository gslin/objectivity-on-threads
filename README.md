# Objectivity on Threads

A browser extension that adds an analysis button to every [Threads](https://www.threads.com/) post, letting you send the post content to an AI chatbot (ChatGPT or Claude) for fact-checking.

一款瀏覽器擴充套件，在每則 [Threads](https://www.threads.com/) 貼文旁加入分析按鈕，將貼文內容送到 AI（ChatGPT 或 Claude）進行事實查核。使用無痕（暫時對話）模式，避免 AI 受到先前對話記憶的影響，確保每次分析都是獨立且客觀的。
